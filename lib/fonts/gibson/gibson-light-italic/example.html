Instructions

1 When clicking the "Download" link (once purchase has been made), choose either "Lite" or "Full" depending upon your preference. Our system will generate the WebFont files automatically and process your download request. You may download one or the other or both options, the choice is up to you. (For the smallest file sizes, we recommend "Lite")
2 Once the Zip file has been downloaded and extracted, you will see a folder named according to your purchased font or complete family. Inside this folder are two folders named "example" and "webfonts." "Example" contains a quick preview of the webfont in various sizes, while "webfonts" contains the generated files.
3 If you do not yet have a CSS directory on your website, create one now. Upload the "webfonts" folder and all the files inside it to your CSS directory. Your structure should look something like this: /css/webfonts/
4The last step is to add the webfont CSS code to your website code, preferably in your header file and after any existing CSS declarations. Copy and paste exactly as follows:
<link rel="stylesheet" href="/css/webfonts/gibson-light-italic.css" type="text/css" charset="utf-8" />
5 To use your font in your site design, simply define the font-family name that is defined in the gibson-light-italic.css on line 18 before the default fonts. For example:
font-family: 'Gibson Light Italic', Helvetica, Arial, Sans-Serif;
That's it! We suggest you choose complimentary default fonts as fallback fonts in your css. For example if you purchased a sans-serif from us, use sans-serif fonts like Helvetica, Arial, Verdana, etc. Visit our YWFT WebFonts page for more details about our WebFonts.
